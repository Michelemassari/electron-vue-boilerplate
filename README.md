# A boilerplate app using Electron & Vue.js

### Usage

```bash
$ npm install -g vue-cli
$ vue init sohelamin/electron-vue-boilerplate my-project
$ cd my-project
$ npm install
$ npm run dev-server
$ npm run dev // on a different tab
```
